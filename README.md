# Real-time-Age-Prediction

## Dependencies
Tested on:
- Windows 10, Python 3.8, PyTorch 1.7.1, TrochVision 0.8.2, CUDA 10.2, cuDNN 7.6

## Usage

### Use trained model for demo
Run the demo script (requires web cam).
```sh
python AP_System.py
```
